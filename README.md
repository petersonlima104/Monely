# 💰 Monely

**Monely** é um aplicativo web para **controle financeiro pessoal**, desenvolvido como **Progressive Web App (PWA)**.
O objetivo do projeto é oferecer uma forma simples e rápida de acompanhar **rendas, despesas e saldo mensal**.

O aplicativo permite que o usuário registre seus ganhos e contas, visualize o saldo em tempo real e acompanhe sua organização financeira mensal.

# 🚀 Funcionalidades

- 🔐 Login com conta Google (Firebase Authentication)
- 💵 Cadastro de rendimentos
- 🧾 Cadastro de contas e despesas
- 📊 Cálculo automático de saldo mensal
- 📅 Controle financeiro por mês
- ♻️ Reutilização automática de contas fixas
- 📱 Aplicativo instalável (PWA)
- ⚡ Interface rápida e responsiva
- ☁️ Armazenamento em nuvem

# 📲 Progressive Web App (PWA)

O Monely pode ser instalado no celular como um aplicativo.

### Android

No navegador Chrome aparecerá a opção:

```
Instalar aplicativo
```

### iOS

No Safari:

```
Compartilhar → Adicionar à tela de início
```
